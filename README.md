# FeiE Printer SDK for Golang
Flying Goose Cloud print go language sdk

## Installation

```bash
go get -u -v github.com/houseme/feie@main 
```

## Usage

```go
package main

import (
    "fmt"
    "github.com/houseme/feie"
)

```
